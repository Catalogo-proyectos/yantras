# Yantras Project

## Overview
Dark luxury aesthetic homepage and web experience for Yantras, built with React, TypeScript, and Three.js for 3D interactive components.

## Frontend Design Standards

This project uses the **frontend-design** skill for all component and page development. Every UI/component task should:
- Generate distinctive, production-grade interfaces
- Maintain high design quality and polish
- Avoid generic AI aesthetics
- Follow the dark luxury aesthetic established in the design system

**Invoke the skill** whenever building or updating:
- React components
- Page layouts
- UI interfaces
- Interactive elements
- Visual styling and design

## Tech Stack
- **React** with TypeScript
- **Three.js / React Three Fiber** for 3D scenes
- **CSS** for styling with dark luxury theme
- **Vite** for build tooling

## Current State
- Dark luxury aesthetic redesign completed
- Hero section with 3D elements
- Dark theme with luxury styling
- Main branch: latest redesign with complete dark aesthetic

## Key Files
- `src/components/Hero3D.tsx` - Main 3D hero component
- `src/index.css` - Global styling and dark theme
- `src/App.tsx` - Application entry point

## When Working on Frontend
- Use `/frontend-design` skill before writing UI code
- Prioritize visual polish and production-grade quality
- Maintain consistency with dark luxury aesthetic
- Test components visually, not just code verification
